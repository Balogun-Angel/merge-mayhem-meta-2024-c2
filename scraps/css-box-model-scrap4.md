There are a variety of ways to customize the margin, border, and padding size based on the top, bottom, left, and right edges. Let's take a look at a few examples in code:

```css
div {
    /* This: */
    margin: 20px;
    /* Is the same as: */
    margin-top: 20px;
    margin-right: 20px;
    margin-bottom: 20px;
    margin-left: 20px;

    /* This: */
    padding: 10px 20px;
    /* Is the same as: */
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;
}
```
